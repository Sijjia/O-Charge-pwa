# Фаза 3 — Partner Essentials (детализация)

> Цель: полный набор функций для партнеров (операторов).

## Функции

- Локации/Станции/Коннекторы: список, статусы, фильтры, поиск
- Аналитика: доходы по периодам, станции, локации; графики
- История пополнений станций/локаций; корректировки/возвраты (если есть)
- Инциденты/здоровье оборудования: uptime/MTBF, быстрые действия (перезапуск/блок)
- Экспорты: CSV/PDF отчеты (доходы/инциденты/сессии)
- RBAC: admin/operator/auditor в рамках организации

## Чеклист

- [ ] Полная иерархия: организация → локации → станции → коннекторы
- [ ] Таблицы со статусами, пагинация, сортировка, фильтры
- [ ] KPI виджеты: количество станций, доход, энергия, uptime
- [ ] Графики доходов (день/неделя/месяц), сравнения
- [ ] Реестр пополнений/листингов по локациям/станциям
- [ ] Экспорт CSV/PDF (проверка форматов)
- [ ] Инциденты: список, фильтры, действия
- [ ] RBAC-интеграция и гейты в UI

## DoD

- [ ] Партнер видит здоровье парка и основные KPI в реальном времени
- [ ] Все таблицы/отчеты фильтруются и экспортируются
- [ ] Действия в UI соответствуют роли
