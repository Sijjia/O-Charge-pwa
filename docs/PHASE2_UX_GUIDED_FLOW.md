# Фаза 2 — Guided Flow “Start charging” и статусы коннекторов

> Цель: пользователь без инструкций проходит зарядку. Добавляем проверки готовности и понятные подсказки.

## Guided Flow

- [ ] Экран проверки перед стартом:
  - [ ] Баланс ≥ минимального порога
  - [ ] Статус коннектора: свободен/исправен/совместим
  - [ ] Станция онлайн (последний хартбит)
  - [ ] Тариф/прогноз стоимости и времени
- [ ] CTA “Начать зарядку” активен только при всех OK
- [ ] Покрытие ошибок: понятные причины и предложенные действия

## Статусы в UI

- [ ] Коннектор: свободен/занят/ошибка/обслуживание
- [ ] Станция: онлайн/оффлайн, последний хартбит
- [ ] Локация: агрегированный статус по станциям/коннекторам

## Интеграция prechecks

- [ ] Хук `useChargingPrechecks` возвращает флаги готовности/причины блокировки
- [ ] Интеграция в `ChargingPage`: отображение статуса/подсказок
- [ ] Без ломающих изменений (поведение по умолчанию как сейчас)

## DoD

- [ ] Пользователь видит, что мешает старту, и как исправить
- [ ] Все проверки корректно обновляются при изменении состояния
- [ ] E2E сценарий “Старт зарядки” проходит с первого раза
